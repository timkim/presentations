PHONEGAP

Early in the 21st century, THE NITOBI CORPORATION advanced mobile development with a tool known as PhoneGap.

PhoneGap was created with a web first mentality. Using HTML, JS, and CSS - developers could make cross platform web applications that launched natively.

Native applications could now be written entirely with web tools using PhoneGap.

This was not called retirement.

It was called evolution.

App Lifecycle

document.addEventListener('deviceready', appInit, false);
document.addEventListener('pause', appPause, false);
document.addEventListener('resume', appResume, false);

var appInit = function() {
    alert('app initd');
}

var appPause = function() {
    alert('paused');
}

var appResume = function() {
    alert('resume');
}

PLUGINS

  • Extend your PhoneGap app with plugins

  • All of the objects in the api are plugins

  • Write native implementation then use JS to call that function

  • Still a work in progress, but check out github.com/phonegap/phonegap-plugins
